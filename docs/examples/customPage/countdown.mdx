## 场景

在我们给员工阅读文件签订协议的时候需要规定员工阅读多长时间才能进行点击确认，然后跳转到其他的页面进行信息填写，有这方面需求可以参考以下案例。

## 实现效果

💡 规定阅读 5 秒才能点击确认跳转

## 效果演示

执行以下操作，实现的效果，如下：

1. 进入页面等待 5 秒才能点击确认按钮；
2. 点击确认后跳转到设置好的网址。

## 操作步骤

### 步骤 1: 创建自定义页面

路径：进入已创建好的应用 >> 页面左上角+号 >> 新建自定义页面

自定义页面除按钮与按钮下方的文本组件，其他组件可按照自身需求配置

### 步骤 4: 按钮点击动作配置

注意：需要将网站地址换成自己想要点击跳转到的地址

### 步骤 5: JS 面板配置

1. 函数内代码配置
   注意：复制代码注意唯一标识的替换
   ![](https://img.alicdn.com/imgextra/i2/O1CN01cw5PPr1TwlusUZnq3_!!6000000002447-2-tps-1918-907.png_.webp)
   代码示例：

```javascript
var that = this;
setTimeout(
  function () {
    that.$('按钮组件唯一标识').set('behavior', 'NORMAL');
    that.$('按钮组件唯一标识').set('loading', false);
  },

  阅读时间的毫秒数
);

var time = 想要多久才能点击确认的秒数;
setInterval(function () {
  if (time > 1) {
    time -= 1;
    that.$('文本组件唯一标识').set('content', '还需阅读' + time + '秒钟');
  } else {
    that.$('文本组件唯一标识').set('content', '点击确认');
  }
}, 1000);
```
