#  使用场景
通过自定义页面实现浏览数量逐次增加并统计的功能。

##  操作步骤
### 步骤一：创建表单和自定义页面
创建一个表单和一个自定义页面，分别是「统计数量中间表」和「需要统计的表单」。
**创建「统计数量中间表」**
创建需要展示统计数量的页面，页面内组件可以根据需要自定义设置。
**操作步骤：**

1. 新建表单页面，命名为「统计数量中间表」。
2. 拖拽两个单行文本组件到中间画布，并在右侧属性栏分别修改组件的名称。
3. 点击保存，表单即可创建成功。

如下所示：
![image.png](https://img.alicdn.com/imgextra/i4/O1CN01IwMJut1HSIAD1GDLF_!!6000000000756-2-tps-1906-888.png)

**创建「需要统计的表单」**
用于存储访问「统计数量中间表」的数据。
**操作步骤：**

1. 新建自定义页面，命名为「需要统计的表单」。
2. 拖拽一个表格组件到中间画布，页面内组件可以根据需要自定义设置。
3. 点击保存，自定义页面即可创建成功。

如下所示：
![image.png](https://img.alicdn.com/imgextra/i1/O1CN01lQEk5Y1LkqfqXtcT9_!!6000000001338-2-tps-1912-886.png)

### 步骤二：配置表单和自定义页面
**自定义页面调用新增表单实例接口**
通过调用接口获取表单的formInstId等相关数据。
点击直接跳转到调用所需接口链接地址：👉 [新增表单实例接口](https://www.yuque.com/yida/support/aql605#oi5tzg)
**操作步骤：**

1. 在「需要统计的表单」中添加远程数据源，名称为 saerchFormDatas 。
2. 在编辑数据源对话框请求地址处填写 `https://www.aliwork.com/dingtalk/web/此处填写您应用的AppType/v1/form/saerchFormDatas.json`。
3. 注意：需要将编辑数据源里面的自动加载按钮关闭（关闭状态为灰色），请求方法为GET。

如下所示： 
![image.png](https://img.alicdn.com/imgextra/i1/O1CN01GDr8yB26YHkhhh6aO_!!6000000007673-2-tps-1912-887.png)

**自定义页面调用更新表单中指定组件值接口**
通过调用接口更新表单的相关数据。
点击直接跳转到调用所需接口链接地址：👉 [更新表单中指定组件值](https://www.yuque.com/yida/support/aql605#vGtpN)
**操作步骤：**

1. 在「需要统计的表单」中添加远程数据源，名称为updateFormData 。
2. 在编辑数据源对话框请求地址处填写 `https://www.aliwork.com/dingtalk/web/此处填写您应用的AppType/v1/form/updateFormData.json`。
3. 注意：需要将编辑数据源里面的自动加载按钮关闭（关闭状态为灰色），请求方法为POST。

如下所示： 
![image.png](https://img.alicdn.com/imgextra/i2/O1CN01yjAPYa1oskMj1Npp7_!!6000000005281-2-tps-1913-882.png)

**自定义页面绑定生命周期函数**
**操作步骤：**

1. 在右侧属性栏中选择绑定动作添加新动作名称为didMount。
2. 点击确定即可。

如下所示：
![image.png](https://img.alicdn.com/imgextra/i3/O1CN01mul2Ik1qw4TAMdDTK_!!6000000005559-2-tps-1905-910.png)



3. 选择属性栏中didMount右侧的&lt;&gt;，定位到当前js代码面板，并将下述代码直接复制进去。

如下所示：
:::danger
注意：1. formUuid为统计数量中间表的formUuid（根据需求可自定义更换表单formUuid）。
   2. 此处textField_kvhrzijz为统计数量中间表中对应的"自定义页面名称"组件唯一标识符（根据需求可自定义更换组件唯一标识）。
:::
![image.png](https://img.alicdn.com/imgextra/i3/O1CN01uQv91d1KPUau1byCb_!!6000000001156-2-tps-1916-895.png)

```javascript
export function didMount() {
  console.log(`「页面 JS」：当前页面地址 ${location.href}`);
  // console.log(`「页面 JS」：当前页面 id 参数为 ${this.state.urlParams.id}`);
  // 更多 this 相关 API 请参考：https://www.yuque.com/yida/support/ocmxyv#OCEXd
  // document.title = window.loginUser.userName + ' | 宜搭';
  const somejson = {
    "textField_kvhrzijz" : "需要统计的表单"
  }
  //此处 textField_kvhrzijz 为统计数量中间表中对应的"自定义页面名称"组件唯一标识符；
  //此处"需要统计的表单"为 统计数量中间表 中提交时"自定义页面名称"对应的名称；
 
  const param ={
    "formUuid":"FORM-9T566BD1REZUKELN04GTL4O9SDAW2KQB8PHVKW7",
    "searchFieldJson" :JSON.stringify(somejson)
  }
  //formUuid为 统计数量中间表 的formUuid
  this.dataSourceMap['searchFormDatas'].load(param).then((res)=>{
    console.log("查看访问次数",res)
    res.data.map(item=>{
      const somejson2 = {
        "textField_kvhp8fjy": Number(item.formData.textField_kvhp8fjy)+ 1
      }
  //此处设置每次加载数据源次数增加一次
      const param_update = {
        "formInstId":item.formInstId,
        "updateFormDataJson": JSON.stringify(somejson2)
      }
      this.dataSourceMap['updateFormData'].load(param_update).then((res) => {
        this.utils.toast({
          title: '加载成功',
          type: 'success',
          size: 'large',
        });
      })
    })
  })
}
```
### 步骤三：新增数据查看效果
**操作步骤：**

1. 在「统计数量中间表」中选择数据管理页面。
2. 点击新增 ，输入自定义页面名称为需要统计的表单。
3. 这里我们需要将 计数里面设置为0，这样可以直观的看出增加的访问次数。
