## 使用场景

如公司每月月底都需要召开全员总结大会，需要提前进行会务准备。会务组成员需要为自己创建日程便于提醒。现在通过宜搭的表单的填写与提交，即可自动为其在钉钉上创建日程。可将待办事项写入到钉钉日历并在钉钉日历中展示。

## 操作步骤

### 步骤 1：表单的创建与配置

创建表单，用于录入日程相关信息，并作为触发连接器功能的数据来源。
**操作步骤：**

1. 新建表单，命名为「日程信息登记」。
2. 在表单内分别添加两个日期组件，分别命名为「日程开始时间」和「日程结束时间」，将格式设置为「年-月-日 时:分」，设置为必填项。（操作如图 2.1-1 所示）

![](https://img.alicdn.com/imgextra/i3/O1CN01R5AYgL1yZBbfGO57o_!!6000000006592-2-tps-1920-952.png_.webp)
图 2.1-1 添加并配置日期组件
:::danger
**注意：**在进行表单数据填写时，应保持**开始时间晚于结束时间**，否则连接器将不会执行。
:::
​

3. 添加一个成员组件，命名为「日程参与人」，开启多选模式，设置为必填项。（操作如图 2.1-2 所示）

![](https://img.alicdn.com/imgextra/i4/O1CN01a2Nld61rpUOUBOWeu_!!6000000005680-2-tps-1920-951.png_.webp)
图 2.1-2 添加并配置成员组件
​

4. 添加一个单行文本组件，命名为「日程标题」，设置为必填项。（操作如图 2.1-3 所示）

![](https://img.alicdn.com/imgextra/i4/O1CN01U54lSu25SxX5YrQ92_!!6000000007526-2-tps-1920-949.png_.webp)
图 2.1-3 添加并配置单行文本组件
​

5. 一个多行文本组件，点击保存，即可。（操作如图 2.1-4 所示）

![](https://img.alicdn.com/imgextra/i1/O1CN01s9Bosv27Rhg9GrMWe_!!6000000007794-2-tps-1920-951.png_.webp)
图 2.1-4 配置多行文本组件并保存表单
​

### 步骤 2：创建集成&自动化

:::info
**拓展阅读：**获取详细的连接器介绍，请移步[集成&自动化-连接器](/docs/guide/concept/connector)
:::
提交表单时，系统自动为指定人员发起日程，因此我们需要配置连接器。
**新建集成&自动化**
**操作步骤：**

1. 后台管理页面>>集成&自动化>>「新建集成&自动化」。（操作如图 2.2-1 所示）

![](https://img.alicdn.com/imgextra/i2/O1CN01iwwRZs1DULTfi00Ef_!!6000000000219-2-tps-2878-1590.png_.webp)
图 2.2-1 连接器入口
​

2. 「新建集成&自动化」对话框>>将连接器命名为「创建日程」>>选择触发类型为「表单事件触发」并指定触发表单>>点击确认。（操作如图 2.2-2 所示）

![](https://img.alicdn.com/imgextra/i4/O1CN01KxyvMB28mbMmsAlPj_!!6000000007975-2-tps-2878-1592.png_.webp)
图 2.2-2 新建连接器
​

**配置集成&自动化**
**（1）配置表单触发事件**
**操作步骤：**
点击选中「表单事件触发」，在触发事件中选择「创建成功」， 数据过滤选择「全部数据」。最后，点击右下角保存按钮，即可。（操作如图 2.2-3 所示）
![](https://img.alicdn.com/imgextra/i1/O1CN01naAQlI1NyhtE0mMKc_!!6000000001639-2-tps-1920-947.png_.webp)
图 2.2-3 配置表单事件触发
​

**（2）配置连接器**
**操作步骤：**

1. 配置「连接器」节点，在选择应用中选中「日程」，并点击下一步按钮。（如图 2.2-4 所示）

![](https://img.alicdn.com/imgextra/i2/O1CN01mDmqmw1DWdSWVLIsy_!!6000000000224-2-tps-1920-949.png_.webp)
图 2.2-4 配置连接器-步骤 1：选择应用
​

2. 在选择执行动作中选中「创建日程」执行动作，点击下一步按钮。（如图 2.2-5 所示）

![](https://img.alicdn.com/imgextra/i4/O1CN0148pief1EeFfByFj4r_!!6000000000376-2-tps-1920-951.png_.webp)
图 2.2-5 配置连接器-步骤 2：选择执行动作
​

3. 在配置执行动作中对各个配置项进行配置，点击保存。（如图 2.2-6 所示）

![](https://img.alicdn.com/imgextra/i4/O1CN01S3V4oN1aXlb79pWsY_!!6000000003340-2-tps-1920-950.png_.webp)
图 2.2-6 配置连接器-步骤 3：配置执行动作
​

4. 点击页面右上角保存后，点击发布即可。

### 步骤 3：配置待办任务内容

通过上述两个步骤，我们实现功能的配置工作已经完成，接下来，只需提交表单即可触发连接器，从而在指定的时间节点为指定的人员创建日程，并在指定的时间节点关闭日程。
**操作步骤：**
填写表单内容，点击提交，即可。（操作如图 2.3-1 所示）
![](https://img.alicdn.com/imgextra/i2/O1CN014WFY1j1sSQ3rmAQDl_!!6000000005765-2-tps-1920-955.png_.webp)
图 2.3-1 录入表单信息并提交
