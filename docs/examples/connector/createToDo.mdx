## 使用场景

用户通过宜搭內的表单填写，实现自动在钉钉内创建钉钉待办消息，发起强有力的消息提醒，做到件件有着落，事事有回音。

## 操作步骤

### 步骤1：表单的创建与配置

创建表单，用于录入待办消息相关信息及触发连接器效果。
**操作步骤：**

1. 创建表单并命名为「待办信息录入」。
1. 添加日期组件，命名为:「发起待办任务时间」并设置其格式为「年-月-日 时:分」，并设置为必填项。（操作如下图所示）

![](https://img.alicdn.com/imgextra/i1/O1CN016heX7L1zrKv4fchz8_!!6000000006767-2-tps-1920-955.png)
​

3. 添加成员组件，命名为「执行人」，并设置为必填项。（操作如下图所示）

![](https://img.alicdn.com/imgextra/i4/O1CN0110L52Y1VWKHLzrtBm_!!6000000002660-2-tps-1920-955.png)
​

4. 添加单行文本组件，命名为「待办标题」，并设置最大长度为 50 且为必填项。（操作如下图所示）

![](https://img.alicdn.com/imgextra/i1/O1CN01SZ6LnP28KCoTjxQpk_!!6000000007913-2-tps-1917-976.png)

:::danger
**说明：**由于连接器配置时，待办标题项最大字数为 50，所以对单行文本组件也进行了最大长度的限制。
:::

5. 添加多行文本组件，命名为「待办描述」，设置为必填项。（操作如下图所示）

![](https://img.alicdn.com/imgextra/i3/O1CN01yyCmuM1MLUMkc7prx_!!6000000001418-2-tps-1909-974.png)
### 步骤2：创建集成&自动化

:::info
**拓展阅读：**获取宜搭连接器相关介绍，请移步[集成&自动化-连接器](/docs/guide/concept/connector)​
:::
提交表单时，系统自动发起待办任务，因此我们需要配置连接器。
**新建集成&自动化**
**操作步骤：**

1. 应用管理后台>>集成&自动化>>「新建集成&自动化」（操作如下图所示）

![](https://img.alicdn.com/imgextra/i3/O1CN01nfZc9J1FDWA6WXScZ_!!6000000000453-2-tps-2878-1594.png)
​

2. 「新建集成&自动化」对话框>>将连接器命名为「待办任务」>>选择触发类型为「表单事件触发」并指定触发表单为「待办信息录入」>>点击确定。（操作如下图所示）

![](https://img.alicdn.com/imgextra/i2/O1CN01K9AsXi22MbBaKFjeW_!!6000000007106-2-tps-2878-1592.png)
​

**配置集成&自动化**
**配置表单触发事件**
**操作步骤：**
点击选中「表单事件触发」，在触发事件中选择「创建成功」， 数据过滤选择「全部数据」。最后，点击右下角保存按钮，即可。（操作如下图所示）
![](https://img.alicdn.com/imgextra/i2/O1CN01JEZrU41tEy2rnbnkj_!!6000000005871-2-tps-1920-955.png)
**配置连接器**
**操作步骤：**

1. 配置「连接器」节点，在选择应用中选中「钉钉待办 1.0」，并点击下一步按钮。（如下图所示）

![](https://img.alicdn.com/imgextra/i1/O1CN01BkzEMk26K5Tgx1Tu8_!!6000000007642-2-tps-1920-955.png)
​

2. 在选择执行动作中选中「创建待办任务」执行动作，点击下一步按钮。（如下图所示）

![](https://img.alicdn.com/imgextra/i2/O1CN01hbcDp81eUnV11IntX_!!6000000003875-2-tps-1920-955.png)

3. 在配置执行动作中对各个配置项进行配置，点击保存。（如下图所示）

![](https://img.alicdn.com/imgextra/i3/O1CN01rz0EyJ1njkycca7YI_!!6000000005126-2-tps-1920-955.png)
​

4. 点击页面右上角保存后，点击发布即可。

### 步骤3：配置待办任务内容

通过上述两个步骤，我们实现功能的配置工作已经完成，接下来，只需提交表单即可触发连接器，从而自动发起待办任务。
**操作步骤：**
填写表单内容，点击提交，即可。（操作如下图所示）
![](https://img.alicdn.com/imgextra/i2/O1CN01LTSfGG1Nf0rhZgXSm_!!6000000001596-2-tps-1920-955.png)
