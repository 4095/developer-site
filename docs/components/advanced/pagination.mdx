## 组件属性

| 属性名   |  属性code  | 类型 | 数据示例 | 默认值  | 动态属性 | 备注 |
| :------ | :------ | :------ | :------ | :----- | :----- | :----- |
| 总记录数 | total | string | | 100 | 是 |  | 
| 当前页码 | current | string | | 1 | 是 |  | 
| 分页尺寸 | size | string | "small" "medium" "large" | "medium" | 否 |  | 
| 分页类型 | type | string |"normal" "simple" "mini" | "normal" | 否 |  | 
| 前进后退按钮样式 | shape | string |"normal" "arrow-only" "arrow-prev-only" "no-border" | "normal" | 否 |  | 
| 每页显示选择器类型 | pageSizeSelector | boolean/string | false, 'filter', 'dropdown'| false | 否 |  | 
| 每页显示选择器可选值 | pageSizeList | string | | "5,10,20" | 否 |  | 
| pageSize | pageSize | string | | 10 | 否 |  | 
| 每页显示选择器在组件中的位置 | pageSizePosition | string | "start" "end"| "end" | 否 |  | 
| 当分页数为1时，是否隐藏分页器 | hideOnlyOnePage | string | | false | 否 |  | 
| 显示跳转输入框与按钮 | showJump | string | | true | 否 |  | 
| 当每页数量改变时 | onPageSizeChange | function | function(pageSize){} || 否 |
| 当页码改变时 | onChange | function | function(currentPagination){} || 否 |
