## 组件属性

| 属性名   |  属性 code  | 类型 | 数据示例  | 动态属性 | 备注 | 
| :------ | :------ |: ------ | :------ | :----- |:----- |
| 时间轴数据 | dataSource  | object  | [{ title: 'Cloudy', time: '2016-06-10 10:30:00' }] | 否 |
| 折叠数据 | fold  | object  | [{ foldArea: [1, 2], foldShow: false }] | 否 |
