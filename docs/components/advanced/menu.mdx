# 菜单


## 何时使用
* 导航菜单是一个网站的灵魂，用户依赖导航在各个页面中进行跳转。一般分为顶部导航和侧边导航，顶部导航提供全局性的类目和功能，侧边导航提供多级结构来收纳和排列网站架构。
## 组件示例
import Iframe from 'components/Iframe'

## 组件属性
| 属性名                         | 属性 Code           | 类型         | 数据示例                                                                 | 备注                                                                                                                         |
| :----------------------------- | :------------------ | :----------- | :----------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------- |
| onItemClick       | onItemClick         | Function     | onItemClick(key, item, event) { console.log(key, item, event);}          |                                                                                                                              |
| 初始打开的子菜单的 key 值      | defaultOpenKeys     | String/Array | ["key1"]                                                                 | ["key1"]                                                                                                                     |
| onOpen | onOpen              | Function     |                                                                  |                                                                                                                              |
| 子菜单打开模式                 | mode                | String       | "inline"                                                                 | 子菜单打开的模式可选值: "inline", "popup"                                                                                |
| 触发行为                       | triggerType         | String       | "click"                                                                  | 子菜单打开的触发行为可选值: "click", "hover"                                                                             |
| 展开内连子菜单的模式           | openMode            | String       | "multiple"                                                               | 展开内连子菜单的模式，同时可以展开一个子菜单还是多个子菜单，该属性仅在 mode 为 inline 时生效可选值: "single", "multiple" |
| 初始选中菜单项的 key 值        | defaultSelectedKeys | String/Array | ["key1-1"]                                                               | ["key1-1"]                                                                                                                   |
| onSelect           | onSelect            | Function     | onSelect(selectedKeys, extraObj) { console.log(selectedKeys, extraObj);} |                                                                                                                              |
| 选择模式                       | selectMode          | String       | "single"                                                                 | 可选值: "single", "multiple","false"                                                                                     |
| 展示方向                       | direction           | String       | "ver"                                                                    | 可选值: "ver", "hoz"                                                                                                     |
| Header 插槽                    | header              |       |                                                                      |                                                                                                           |
| Footer 插槽                    | footer              |       |                                                                      |                                                                                                          |
| 是否自动获得焦点               | autoFocus           |              |                                                                          |                                                                                                                              |
