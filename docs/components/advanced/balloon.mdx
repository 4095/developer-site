## 组件属性

| 气泡类型 | TYPE | string |"balloon" "tooltip" | "balloon" | 是 |  | 
| 样式类型 | type | string |"normal" "primary" | "normal" | 是 |  | 
| 弹窗内容 | content | string | | "提示内容" | 否 |  | 
| 触发行为 | triggerType | string | "hover" "click" "focus" | "click" | 是 |  | 
| 关闭按钮 | closable | string | | true | 是 |  | 
| 默认显示 | defaultVisible | string | | false | 是 |  | 
| 延时显示 | delay | string | | 0 | 否 |  | 
| 弹出方向 | align | string | "t" "b" "l" "r" "tl" "tr" "bl" "br" "lt" "lb" "rt" "rb" | "b" | 是 |  | 
| 占据空间 | display | string | "inline-block" "block"| "inline-block" | 是 |  | 
| 最大宽度 | overlayMaxWidth | string | | "300px" | 是 |  | 
| 显示弹层 | balloonOverlayVisible | string | | true | 否 |  |
