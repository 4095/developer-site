## 组件属性

| 属性名   |  属性code  | 类型 | 数据示例  |  备注 |
| :------ | :------ | :------ | :------ | :----- |
| 步骤数据 | dataSource  | object  | [{ title: 'step1', content: 'Open' }, { title: 'step2', content: 'Put'}] |  |
| 当前步骤 | current  | number  | 1 |  |
| 类型 | shape  | string  | 'circle' | 'circle', 'arrow', 'dot' |
| 展示方向 | direction  | string  | 'horizontal'  | 'horizontal', 'vertical' |
| 横向布局时的内容排列 | labelPlacement  | string  |  'vertical' | 'vertical', 'horizontal' |
| 只读模式 | placeholder  | bool  |  false |  |
| 开启动效 | animation  | bool  |  true |  |
| 点击按钮动作 | onClick  | func  |  'function onClick(index) {}' |  |
