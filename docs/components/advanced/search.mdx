# 搜索


## 何时使用
* 页面、表单数据搜索时使用。
## 组件示例
import Iframe from 'components/Iframe'

<Iframe url="https://www.aliwork.com/developer/search-v2?isRenderNav=false" />

## 组件属性

| 属性名             | 属性 Code          | 类型     | 数据示例 | 备注                                          |
| :----------------- | :----------------- | :------- | :------- | :-------------------------------------------- |
| 大小               | size               | String   | "medium" | 可选值: "medium","large"                      |
| 类型               | type               | String   | "normal" | 可选值: "normal","primary","secondary","dark" |
| 形状               | shape              | String   | "normal" | 可选值: "normal","simple"                     |
| onChange           | onChange           | Function |          | 输入关键字时的回调                            |
| onSearch           | onSearch           | Function |          | 点击搜索按钮触发的回调                        |
| 选择器默认值       | defaultFilterValue | String   |          |                                               |
| 选择器             | filter             | Array    |          |                                               |
| 选择器值           | filterValue        | String   |          |                                               |
| onFilterChange     | onFilterChange     | Function |          | 选择器发生变化时回调                          |
| 搜索框下拉联想列表 | dataSource         | Array    |          |                                               |
| 默认提示           | placeholder        | String   |          |                                               |
| 按钮内容           | searchText         | String   |          |                                               |
| 是否显示清除按钮   | hasClear           | Boolean  | false    | 可选值: false,true                            |
| 是否禁用           | disabled           | Boolean  | false    | 可选值: false,true                            |
