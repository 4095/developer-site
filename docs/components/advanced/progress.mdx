## 组件属性

| 属性名   |  属性code  | 类型 | 数据示例 | 动态属性 | 备注 |
| ------ | ------ | ------ | ------ | ----- | ----- |
| 尺寸 | size  | string  | "large" | 否 |  可选值："small", "middle", "large" |
| 形态 | shape  | string  | "line" | 否 | 可选值："line", "circle" |
| 百分比 | percent  | number  | 10  | 是 | 支持 0 到 100，超出范围，渲染不正常 |
| 状态 | state  | string  | "normal" | 否 | 可选值："normal", "success", "error" |
| 边框 | hasBorder  | bool  | false  | 否 | 是否含有边框，只有条形支持 |
| 色彩阶段变化模式 | progressive  | bool | false | 否 | 颜色随数字变化 |
