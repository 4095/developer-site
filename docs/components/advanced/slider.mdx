## 组件属性

| 属性名   |  属性code  | 类型 | 数据示例  |  备注 |
| :------ | :------ |: ------ | :------ | :----- |
| 轮播项   | images | object | {src: 'https://test.png', title: 'title'} |  |
| 类型 | type  | string  | 'single', 'multi'  | 轮播方式, 单图, 多图|  |
| 图片宽度 | slideImageWidth  |  string  | '100%'  |   |
| 图片高度 | slideImageHeight  |  string  | '300px'  |   |
| 图片间距 | margin  |  number  | 10 |   |
| 同时展示的图片数量 | slidesToShow  |  number  | 2 |   |
| 同时滑动的图片数量 | slidesToScroll  |  number  | 1  |   |
| 居中模式 | centerMode  |  bool  | false  |   |
| 多图轮播时，点击选中后自动居中 | focusOnSelect  |  bool  | false  |   |
| 轮播方向 | slideDirection  |  string  | 'hoz' | 'hoz' , 'ver' |
| 轮播速度 | speed  |  number  | 500  |  单位: ms |
| 动效类型 | animation  |  string  | 'slide'  |  'slide', 'fade' |
| 箭头 | arrows  |  bool  | true  |   |
| 箭头大小 | arrowSize  |  string  | 'medium'  |  'medium', 'large' |
| 箭头位置 | arrowPosition  |  string  | 'inner'  |  'inner', 'outer' |
| 箭头方向 | arrowDirection  |  string  | 'hoz'  | 'hoz', 'ver' |
| 自动播放 | autoplay  |  bool  | false  |  |
| 自动播放的速度 | autoplaySpeed  |  number  | 3000  |  |
| 导航锚点 | dots  |  bool  | true  |  |
| 导航锚点位置 | dotsDirection  |  string  | 'hoz'  |  |
| 锚点导航触发方式 | triggerType  |  string  | 'click  | 'click', 'hover' |
