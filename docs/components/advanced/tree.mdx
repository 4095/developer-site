| :------ | :------ | :------ | :------ | :----- | :----- |
| 数据  | dataSource | Array | [{"key":"0-0","label":"0-0","children":[{"key":"0-0-0","label":"0-0-0","children":[{"key":"0-0-0-0","label":"0-0-0-0","children":[{"key":"0-0-0-0-0","label":"0-0-0-0-0"}]},{"key":"0-0-0-1","label":"0-0-0-1"}]},{"key":"0-0-1","label":"0-0-1","children":[{"key":"0-0-1-0","label":"0-0-1-0"},{"key":"0-0-1-1","label":"0-0-1-1"}]}]}] | 是 | JSON 数组 |
| 显示线 | showLine | Bool | false | 否 |  |
| 多选 | multiple | Bool | false | 否 |  |
| 支持编辑 | editable | Bool | false | 否 |  |
| 支持拖拽 | draggable | Bool | false | 否 |  |
| 默认展开 | defaultExpandAll | Bool | true | 否 |  |
| 默认展开节点数组 | defaultExpandedKeys | Array | [] | 是 |  |
| 支持选中 | selectable | Bool | true | 否 |  |
| 默认选中节点 | selectedKeys | Array | [] | 是 |  |
| 复选框 | checkable | Bool | false | 否 |  |
| 默认勾选节点 | checkedKeys | Array | [] | 否 |  |
| 复选框完全受控 | checkStrictly | Bool | false | 否 |  |
| 定义选中时回填的方式 | checkedStrategy | String | 'parent' | 否 |  |
| 开启异步加载 | isLoadData | Bool | false | 否 |  |
| loadData 函数 | loadData | String | function loadData(data){ return new Promise((resolve) => { resolve(); })} | 否 |  |
| 节点是否占满剩余空间 | isLabelBlock | Bool | false | 否 |  |
| 节点是否占满一行 | isNodeBlock | Bool | false | 否 |  |
| 选中或取消选中节点时 | onSelect | Func | function(ctx, selectedKeys, extra) { console.log('选中或取消选中节点时'); } | 否 |  |
| 勾选或取消勾选复选框时 | onCheck | Func | function(ctx, checkedKeys, extra) { console.log('勾选或取消勾选复选框时'); } | 否 |  |
| 展开收起节点时 | onExpand | Func | function(ctx, expandedKeys, extra) { console.log('展开收起节点时'); } | 否 |  |

| 拖拽节点在目标节点上移动时 | onDragOver | Func | function(ctx, info) { console.log('拖拽节点在目标节点上移动时'); } | 否 |  |
| 拖拽节点离开目标节点时 | onDragLeave | Func | function(ctx, info) { console.log('拖拽节点离开目标节点时'); } | 否 |  |
