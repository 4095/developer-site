# 选项卡
选项卡切换组件。

## 何时使用
TAB 让用户可以在不同子任务、视图、模式之间切换，它具有全局导航的作用，是全局功能的主要展示和切换区域，一个TAB标记一个核心功能，TAB之间可以快速点击切换。该窗口包含2个以上的选项卡，所有选项卡可以排列在一行中，即使该用户界面被本地化后也是如此。提供平级的区域将大块内容进行收纳和展现，保持界面整洁。

宜搭提供了三级选项卡，分别用于不同的场景。

* 普通选项卡，引领整页面的内容，起导航的作用。
* 文本型选项卡。
* 包裹型选项卡，在页面结构中，只是局部展示，需要和其他内容结合出现。
* 胶囊型选项卡。

## 组件属性
| 属性名   |  属性code  | 类型 | 数据示例  | 动态属性 | 备注 |
| ------ | ------ | ------ | ------ | ----- | ----- |
| 选项 | items | Array  | [{ disabled: false, primaryKey: "2", title: "标签项 2" }] | 否 | 可以设置默认激活选项、禁用选项和标签名 |
| 形态 | shape | String | 普通型 | 否 | 可选项有普通型，包裹型，文本型，胶囊型等 |
| 大小 | size | String | 正常 | 否 | 可选项有小，正常 |
| 选项卡过多时的滑动模式 | excessMode | String | 滑动 | 否 | 可选项为滑动、下拉 |
| 导航选项卡的位置 | tabPosition | String | 上 | 否 | 可选项为上、下、左、右 |
| 选项卡切换时 | onChange | function | `function onChange(activeIndex, key){ console.log(activeIndex, key); }` | 否 |  |
