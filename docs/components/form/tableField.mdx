# TableField 明细

## 何时使用

- 用于子表单提交场景；

## 类型定义
### 自定义操作项配置结构

```ts
interface Action {
  content: string; // 操作项文案
  callback: ({index: number, groupId: string, itemValue: any, actionKey: string }) => void; // 点击回调
  render: ({index: number, groupId: string, itemValue: any, actionKey: string }) => ReactNode; // 自定义渲染
}
```
