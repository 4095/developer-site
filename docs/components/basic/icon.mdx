## 组件属性

| 属性名 |  属性 code  | 类型 | 数据示例  | 动态属性 | 备注 |
| :------ | :------ | :------ | :------ | :----- | :----- |
| 图标   | src | string | 'smile' | 否 |
| 大小   | size | string | 'medium' | 否 | 从小到大依次为: `xxs` , `xs` , `small` , `medium` , `large` , `xl` , `xxl` |

填充色请直接修改样式中的颜色属性。
