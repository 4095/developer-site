## 组件属性

  | 属性名 |  属性code  | 类型 | 数据示例  | 动态属性 | 备注 |
| :------ | :------ | :------ | :------ | :----- | :----- |
| 标题   | content | string | “按钮” | 否 |
| 按钮类型 | type  | string  | “primary”  | 否 | 可选值："primary", "normal", "secondary", "ghostLight", "ghostDark", "warningNormal","warningPrimary", "textNormal", "textPrimary", "textSecondary"  |
| 按钮尺寸 | size  | string  | medium  | 否 | 可选值："small", "medium", "large"|
| 状态设置 | behavior | string  | "NORMAL"  | 否 | 可选值："NORMAL", "DISABLED", "HIDDEN"|
| 图标 | iconName | string  | “add”  | 否  | 
| 加载状态 | loading  | boolean  | false | 否 | 仅 PC 端支持 | 
| 加载点击 | triggerEventsWhenLoading | boolean  | false  | 否 |
| 点击时事件 | onClick | function | function(){ console.log('click'); } | 否 |
