# Link 链接

跳转超链接，用于跳转到新页面。

## 何时使用

- 切换访问当前应用的其他页面；
- 跳转到外部页面。

## 组件示例

import Iframe from "components/Iframe";

<Iframe url="https://www.aliwork.com/developer/link-v2?isRenderNav=false" />
