## 组件属性

| 属性名 |  属性code  | 类型 | 数据示例  | 动态属性 | 备注 |
| :------ | :------ | :------ | :------ | :----- | :----- |
| 标题   | title | string | "标题" | 是 |
| 宽度 | width  | string  | "520px"  | 否 |
| 高度 | height  | string  | "600px"  | 否 |
| 是否显示 | visible  | bool  | false  | 是 |
| 显示遮罩 | hasMask | bool  | true  | 否 |
| 弹出位置 | placement | string  | "top"  | 否 | 可选值："top","bottom","left","right" |
| 关闭方式 | closeable | array  | ["mask"]  | 否  | 可选值："mask","esc" |
| 显示footer | footer | bool  | true  | 否 |
| footer对齐方式 | footerAlign | string  | "right"  | 否 | 可选值："left","right","center" |
| footer排列方式 | footerActions | string  | "ok,cancel"  | 否 | 可选值："ok,cancel","cancel,ok","ok","cancel" |
| onOK | onOK | function  | ()=>{}  | 否 |
| onCancel | onCancel | function  | ()=>{}  | 否 |
| onClose | onClose | function  | ()=>{}  | 否 |
| afterClose | afterClose | function  | ()=>{}  | 否 |
