| 属性名 |  属性code  | 类型 | 数据示例  | 动态属性 | 备注 |
| :------ | :------ | :------ | :------ | :----- | :----- |
| 图片地址   | src | string | 图片资源地址 | 否 |  |
| 宽  | height | number | 300 | 否 |  |
| 宽度 auto   | autuWidth | number | 300 | 否 |  |
| 高   | height | number | 300 | 否 |  |
| 高度 auto   | autuHeight | number | 300 | 否 |  |
| 摆放方式  | fit | enum | ‘contain’ | 否 |  |
| 圆角   | round | string | ‘8px’ | 否 | |
| 圆角大小 | roundRadius | string | ‘8px’ | 否 | |
| title  | title | string | ’‘ | 否 | 标题 |
| 替代文本  | alt | string | ’‘ | 否 | 图片未加载时的占位文字 |
| 点击事件  | onClick | function |
